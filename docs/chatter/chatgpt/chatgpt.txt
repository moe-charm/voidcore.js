セットアップと実行方法

このリポジトリは、ブラウザ上で完結する構成になっています。サーバー環境は不要ですが、ローカルでテストする場合は、以下のように http-server を使うことができます。
🌐 ローカルでデモを開く方法

# 1. Python の簡易HTTPサーバーを使う場合:
python3 -m http.server

# または Node.js を使っている場合:
npx serve .

ブラウザで以下にアクセスしてください：

http://localhost:8000/examples/index.html

🎯 プロジェクトの目的・ビジョン

VoidCore.js は、完全に自律する存在としてのプラグインを、数行の JavaScript だけで表現できることを証明するために開発されました。

この仕組みは、従来のプラグインマネージャーやイベントドリブンシステムとは異なり、

    コアは一切命令しない

    プラグインは自ら掲示板に登録し、自ら観測し、自ら終了する

    状態遷移や依存解決も、プラグイン自体の責任で処理される

という、**「完全自律存在モデル」**を実現します。

この思想とアーキテクチャは、ブラウザだけでなく、組み込み・分散システム・データフロー処理など、幅広い応用可能性を持っています。
